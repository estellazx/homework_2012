Select a category from the menu at left to browse the catalogue of animations by category. Most animations are 512 by 384 pixel avi files and are generally several megabytes long. Small screenshots are provided, and the user may view the animation in three ways: direct link to the media file, and two versions of a link to a page with the animation embeded in a web page with a brief comment on the animation. There are two methods used to embed the animation because not all work equally well with any one method. We recommend using the Firefox browser with quicktime. Right-click on the media file link to download and save an animation to your own computer.
